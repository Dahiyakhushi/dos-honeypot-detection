DoS Attack Detection and Monitoring using Honeypot System
Project Overview

Denial of Service (DoS) attacks disrupt network services by flooding systems with malicious traffic. This project implements a Honeypot-based DoS Detection and Monitoring System that attracts attackers, monitors suspicious activities, logs attack data, and generates alerts. The system helps in analyzing attack behavior and improving cybersecurity awareness.

Problem Statement

Traditional firewalls are not capable of detecting all types of DoS attacks. Systems often lack real-time monitoring, proper logging, and analysis of attacker behavior. Hence, there is a need for an intelligent system that can effectively detect, monitor, and analyze DoS attacks.

Technology Stack Used

Software:

Python

Flask / FastAPI

HTML, CSS, JavaScript

MongoDB / Firebase
Expected Outcome

Detection of abnormal traffic

Identification of attacker IP addresses

Logging and analysis of DoS attack patterns

Improved system security monitoring
